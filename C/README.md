# React Form Validation And Login App (Vite)

A simple and responsive **React + Vite** application demonstrating form validation and user login functionality.  
The project includes a sample user data file (`utils/users.js`) for testing login verification.

---

## Features

- Built with **React + Vite**
- Fully **responsive** design
- Client-side **form validation**
- Includes **sample user login details** for testing
- Organized folder structure (components, utils)

---

## Tech Stack

- **Frontend:** React (with Vite)
- **Language:** JavaScript / JSX
- **Styling:** CSS (responsive layout)
- **Build Tool:** Vite

---

## Getting Started

Follow these steps to clone and run the app locally

### Project Structure

vite-react-form-validation/\
├── public/\
├── src/\
│ ├── components/\
│ ├── utils/\
│ │ └── users.js # Sample users data\
│ ├── App.jsx\
│ ├── main.jsx\
│ └── index.css\
├── package.json\
└── vite.config.js

### Clone the Repository

git clone https://github.com/kishore906/Solum-Assignment.git

### Navigate into the Project Folder

cd <folder-name (which is C folder)>

### Install Dependencies

npm install

### Start the Development Server

npm run dev

### Open App in the Browser

http://localhost:5173/

### Test Login Credentials

You can find sample users credentials inside:\
src/utils.users.js\
Use these credentials in the login form to test validation and verification
