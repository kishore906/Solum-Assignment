# Java Program

This folder contains Java program written and executed using **IntelliJ IDEA**, but they can also be run using **Eclipse** or the **Command Line**.

---

## Requirements

- **Java JDK 17+** (or the version your program uses)
- Any Java IDE such as:
  - [IntelliJ IDEA](https://www.jetbrains.com/idea/)
  - [Eclipse](https://www.eclipse.org/downloads/)
  - or simply the **Java command line tools**
- Works on **Windows**, **macOS**, and **Linux**

---

## How to Run the Program

### Option 1 — Run in IntelliJ IDEA

1. Open IntelliJ IDEA
2. Click **File → Open...** and select your project folder
3. Wait for IntelliJ to load dependencies (if any)
4. Open the `.java` file containing the `main()` method
5. Click the green ▶️ **Run** button beside `main()`

---

### Option 2 — Run in Eclipse

1. Open Eclipse IDE
2. Go to **File → Import → Existing Projects into Workspace**
3. Select your project folder
4. Once imported, right-click the file with the `main()` method
5. Choose **Run As → Java Application**

---

### Option 3 — Run from Command Line (Manual Way)

#### Navigate to the folder containing `.java` file - here it is A folder

```
cd path/to/java/file # cd A
javac MysticWaves.java
java MysticWaves

```
